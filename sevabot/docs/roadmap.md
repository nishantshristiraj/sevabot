# SevaBot Roadmap

## Phase 1 – SF MVP (DONE)
- Crawl cgisf.gov.in, VFS SF
- Build KG JSON
- WhatsApp 3-question flow

## Phase 2 – All-US (Week 2)
- Add NY, Chicago, Houston, Atlanta, DC KGs
- Zip→consulate router
- Test with 5 users per consulate

## Phase 3 – Live Crawler (Week 3)
- GitHub Action daily 6am PT
- Parse fees/timelines with BeautifulSoup
- Slack alert on diff

## Phase 4 – Personalizer (Week 4)
- Jinja templates for Annexure E/D
- Photo validator (35x45mm, 600dpi)
- PDF checklist generation

## Phase 5 – Web (Week 5)
- /checklist/{id} shareable
- Status tracker integration with VFS

## Phase 6 – Proactive (Week 6)
- Store passport expiry
- WhatsApp reminders at 9mo, 6mo, 1mo
